# Personal Library Scanner

Monorepo for a small personal library app:

- `backend/`: FastAPI backend, database models, OCR/AI pipeline
- `mobile/`: Expo mobile client
- `docs/`: architecture and API notes
- `work/`: handoff and tutor-mode continuity

## Current Goal

Bootstrap a backend-first vertical slice that can be tested immediately:

1. start the API
2. hit a health route
3. add book and scan flows next

## First Milestone

- FastAPI app skeleton
- testable health endpoint
- dependency manifest
- backend test harness
